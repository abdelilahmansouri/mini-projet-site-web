Calcul de la contrainte maximale d'une poutre

Posté par Hassan · Mécanique

Question : Comment définir la contrainte maximale sur une poutre simplement appuyée avec charge centrale ?

Réponse : Utiliser la formule σmax = Mmax * c / I avec Mmax = P * L / 4.

Moteur pas à pas 28BYJ-48 qui vibre

Posté par Sarah · Électronique

Question : Le moteur vibre mais ne tourne pas ; quelle est la cause possible ?

Réponse : Mauvaise séquence de bobines. Utiliser : A → AB → B → BC → C → CD → D → DA.

Différence convection naturelle / forcée

Posté par Mehdi · Thermique

Réponse : Naturelle = flottabilité ; Forcée = ventilateur/pompe.
